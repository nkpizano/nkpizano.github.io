All About the Basics: R and Rstudio

Oct 8th, 2025 posted by N.K.Pizano

Installing R and Rstudio Windows

  • Download R from the CRAN website
  • Run the .exe file that was just downloaded
  • Go to the Rstudio download page located on posit
  • Under Installers select Windows Vista 10/11 - RSTUDIO-xxxx.yy.z-zzz.exe (where x = year, y = month, and z represent version numbers)
  • Double click the file to install it
  • Once it’s installed, open RStudio to make sure it works and you don’t get any error messages

      • Installing R and Rstudio MacOS

      • Download R from the CRAN website
      • Select the .pkg file for the latest R version
      • Double click on the downloaded file to install R
      • Install XQuartz website
      • Go to the Rstudio download page located on posit
      • Under Installers select Mac OS 13+ - RSTUDIO-xxxx.yy.z-zzz.dmg (where x = year, y = month, and z represent version numbers)
      • Double click the file to install it
      • Once it’s installed, open RStudio to make sure it works and you don’t get any error messages

      What are Rmarkdowns

      RMarkdown (.Rmd) files combine text, code, and output (like plots or tables) in one document. They’re perfect for:

      • Reproducible reports
      • Class notes
      • Interactive documents
      You can knit an RMarkdown file into HTML, PDF, or Word to produce polished documents that mix narrative and analysis seamlessly. Need help starting a Rmarkdown? Check out the following resources:
      • Rmarkdown cheatsheet
